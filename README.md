# icml26

This repository contains the code used to construct the ROAMM dataset and to implement the mind-wandering (MW) classifier and EEG-to-text decoding pipelines. This is a minimal release; a complete and fully documented version of the repository will be made available upon paper acceptance.

EEG-to-text decoding scripts adopted from Wang & Ji, 2024: https://github.com/MikeWangWZHL/EEG-To-Text
